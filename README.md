# `Banking operations widget`

Программа создана для фильтрации и сортировки банковских счетов по дате и оплате.

## Project dependencies:
- The program uses the version Python 3.12
- flake8 = "7.1.0"
- black = "24.4.2"
- isort = "5.13.2"
- mypy = "1.11.0"
